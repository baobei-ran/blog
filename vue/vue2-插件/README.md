# vue插件大全


1. 图片实现裁剪功能

```base
npm install vue-img-cutter --save
```


2. Vue中图片压缩的工具 ```image-conversion```

```base
npm i image-conversion --save
```


3. vue 图片放大查看

```base
npm install v-viewer --save

```
